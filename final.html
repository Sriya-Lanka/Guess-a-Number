Welcome to the Guessing Game!

This exciting game challenges you to guess a number between 1 and 100. You have 10 chances to guess the correct number. It's a fun and interactive way to test your intuition and numerical skills. Experience the thrill of guessing, the joy of getting it right, and the suspense as you approach your last chances. We hope you enjoy this simple yet engaging game created just for you!

Development Details:

We used HTML to create the structure of the game, CSS to style it visually, and JavaScript to add interactivity and game logic. Here's a breakdown of how we used each technology:

HTML: For defining the main elements of the game, such as the headings, paragraphs, buttons, and game interface. CSS: For styling the game's appearance, including colors, fonts, spacing, and layout. JavaScript: For handling user interactions, generating random numbers, checking guesses, providing feedback, and managing the overall game flow.
